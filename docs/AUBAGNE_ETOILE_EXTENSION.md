# Extension opportuniste — Pays d’Aubagne / Étoile / Marseille Est

## Objectif

Cadrer une extension SEO/GEO opportuniste autour du Pays d’Aubagne, de l’Étoile et de Marseille Est, sans diluer le positionnement principal du site : **Provence Verte & Verdon**.

Cette extension ne doit pas devenir un second territoire principal. Elle doit servir uniquement si elle apporte :

- des opportunités vendeurs crédibles ;
- un angle éditorial différenciant ;
- un lien clair avec l’expérience et le réseau d’Alexandre Lopez ;
- une capacité de conversion vers l’avis de valeur ou le contact.

## Décision de positionnement

### Positionnement principal à conserver

Le socle reste :

```plain text
Provence Verte & Verdon
→ Barjols / Cotignac / Lorgues / Brignoles / villages du Var intérieur
→ avis de valeur vendeur
→ biens de caractère / maisons / marché local
```

### Extension secondaire assumée

L’extension doit être présentée comme :

```plain text
Interventions ciblées côté Pays d’Aubagne, Étoile et Marseille Est
→ uniquement lorsque le projet correspond au réseau, à l’expérience ou au potentiel vendeur
```

Ne pas remplacer les pages Provence Verte par une promesse généraliste type “immobilier Provence-Alpes-Côte d’Azur”.

## Recherche rapide SERP

### Aubagne

Les résultats observés montrent un marché concurrentiel, déjà bien occupé par agences et portails de prix.

Repères extraits des SERP :

- PAP : prix moyen tous biens autour de 3 076 €/m² ; maisons autour de 4 625 €/m².
- Meilleurs Agents : moyenne tous biens autour de 3 739 €/m².
- SeLoger : moyenne autour de 3 824 €/m² ; maisons autour de 3 906 €/m² selon extrait.
- King Immobilier : maisons autour de 4 922 €/m², appartements autour de 2 809 €/m².
- Paradissimmo : appartements autour de 3 131 €/m², maisons autour de 3 350 €/m².

**Lecture :** forte variation selon quartiers, type de bien, accès, extérieurs, stationnement et proximité collines / centre / axes.

### Pays de l’Étoile — La Destrousse / Roquevaire / Auriol / Peypin

Résultats observés :

- Netvendeur affiche des repères élevés sur les communes proches : Roquevaire, Peypin, Auriol, La Bouilladisse autour de 3 700 à 4 200 €/m² selon extrait.
- SeLoger : La Destrousse autour de 3 614 €/m² avec fourchette large.
- Efficity : La Destrousse maisons autour de 3 940 €/m².
- Présence d’agences locales très ancrées, notamment sur le Pays de l’Étoile.

**Lecture :** secteur périurbain résidentiel, fortement lié à la maison familiale, accès autoroute, écoles, calme, terrain et proximité Aubagne / Marseille.

### Marseille Est — 11e / 12e

Résultats observés :

- Meilleurs Agents : Marseille 11e autour de 3 820 €/m² tous biens.
- Meilleurs Agents Marseille global : maisons autour de 4 654 €/m², avec forte fourchette.
- Figaro : Marseille 11e maisons autour de 3 931 €/m² médian.
- Efficity : Marseille 12e maisons autour de 4 630 €/m².
- Netvendeur rappelle que certains quartiers de l’Est marseillais gardent une logique d’anciens villages : La Treille, Saint-Barnabé, Saint-Julien, Montolivet, etc.

**Lecture :** marché très hétérogène et concurrentiel ; intéressant uniquement sur un angle “maisons / villas / quartiers Est / village dans la ville”.

## Angle éditorial recommandé

### Ne pas dire

Éviter :

- “Alexandre Lopez spécialiste d’Aubagne” si ce n’est pas encore prouvé ;
- “expert Marseille Est” sans preuve forte ;
- pages génériques copiées depuis Provence Verte ;
- promesse de couverture large.

### Dire plutôt

Utiliser un wording plus crédible :

- “Interventions ciblées côté Pays d’Aubagne / Étoile / Marseille Est” ;
- “lorsque le projet correspond au réseau et à la méthode d’accompagnement” ;
- “lecture vendeur : positionnement prix, préparation de la mise en vente, comparaison des biens similaires” ;
- “maisons, villas, biens avec extérieur, secteurs résidentiels”.

## Structure recommandée si production de pages

Route : `/marche/pays-aubagne-etoile`

Rôle : cadrer l’extension sans la mettre au même niveau que Provence Verte.

Sections :

2. Les secteurs à considérer.
3. Ce qui fait varier le prix d’une maison.
4. Quand demander un avis de valeur.
5. CTA vers `/avis-de-valeur-immobilier` et `/outils/vendre`.

### Pages locales éventuelles

Prioriser seulement si le tracking ou les opportunités justifient la production :

1. `/marche/aubagne`
2. `/marche/la-destrousse`
3. `/marche/roquevaire`
4. `/marche/auriol`

Chaque page doit inclure :

- title/meta uniques ;
- repères de prix prudents ;
- typologies de biens ;
- facteurs d’estimation ;
- FAQ locale ;
- maillage vers l’avis de valeur ;
- mention claire que le territoire principal reste Provence Verte & Verdon.

## Conditions de lancement

Ne pas produire tout de suite une grappe complète.

Lancer une première page seulement si au moins une condition est remplie :

1. Un lead réel arrive sur ce secteur.
2. Une opportunité commerciale est identifiée.
3. Search Console montre déjà des impressions liées à Aubagne / Étoile / Marseille Est.
4. Alexandre souhaite explicitement pousser cette zone.
5. Un contenu différenciant peut être produit sans promesse excessive.

## Garde-fous SEO / marque

- Ne pas modifier le H1 de la page `/marche` pour inclure Aubagne / Marseille Est.
- Ne pas mettre Aubagne au même niveau que Barjols / Cotignac / Lorgues dans le discours principal.
- Ne pas ajouter toutes les communes au sitemap tant qu’elles ne sont pas enrichies.
- Ne pas créer de pages locales pauvres uniquement pour indexation.
- Garder les CTA vers l’avis de valeur, pas vers une promesse de mandat immédiat.

## Roadmap proposée

### Étape 1 — Cadrage livré

- Documenter le territoire secondaire.
- Définir secteurs, angles, conditions.
- Ne pas produire de pages publiques encore.

### Étape 2 — Signal marché

Pendant 30 à 60 jours :

- observer Search Console ;
- regarder les clics CTA via tracking ;
- noter les leads entrants hors Provence Verte ;
- regarder si Aubagne / Étoile apparaît dans les requêtes.

### Étape 3 — Première page test

Si signal positif :

- créer une page hub `/marche/pays-aubagne-etoile`, ou
- créer directement `/marche/aubagne` si le signal est clairement sur Aubagne.

### Étape 4 — Extension contrôlée

Ajouter ensuite seulement :

- La Destrousse ;
- Roquevaire ;
- Auriol ;
- puis éventuellement une micro-zone Marseille Est, plutôt qu’un arrondissement entier.

## Recommandation finale

Statut recommandé : **cadré, mais ne pas produire immédiatement**.

Le site vient de renforcer son socle SEO/GEO Provence Verte & Verdon. Il vaut mieux laisser le tracking accumuler les premiers signaux avant de produire cette extension. La prochaine action prioritaire devrait plutôt être :

1. surveiller les premiers signaux SEO / conversion ;
2. enrichir Brignoles ou Pontevès si l’on veut continuer la grappe Provence Verte ;
3. créer un article vendeur fort si l’on veut accélérer l’autorité éditoriale.
